# [웹 백엔드 - ERP] GIANT v1.0

## 기술 스택
- Backend: Spring Boot
- Database: PostgreSQL
- Language: Kotlin(jdk-17)

## 실행 하기

### 1. GIT CLONE
```bash
git clone https://github.com/cw7430/giant-back.git
```

### 2. PostgreSQL 17 설치
- <https://www.postgresql.org/download/>

### 3. application-local.yml 작성
- application-local-example.yml 참고

### 4. 실행하기
- InteliJ, Eclipse, VSCode 등에서 실행 가능
